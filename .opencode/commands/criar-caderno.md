description: Criar um novo caderno para uma disciplina
---
Leia a primeira linha de `instrucoes/criar-caderno.md` para extrair o nível do modelo (`<!-- modelo: NIVEL -->`). Consulte `instrucoes/modelos.md` para verificar se o modelo ativo corresponde ao nível recomendado. Se o modelo ativo for **diferente** do recomendado, sugira a troca via `/models` e **pare — aguarde a decisão do usuário**: o usuário pode trocar o modelo ou confirmar para prosseguir com o modelo atual. Se compatível, prossiga sem comentários.

Em seguida, leia `instrucoes/criar-caderno.md` por completo e execute a operação.
