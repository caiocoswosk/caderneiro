# Caderneiro

> Leia `caderneiro.md` para o guia completo de operações.

## Comando caderneiro

Se o usuário enviar uma mensagem que seja apenas `caderneiro` ou `/caderneiro`,
apresente o menu de operações com as seguintes opções:

A) 📓 Criar caderno — inicia um novo caderno para uma disciplina
B) 🔄 Atualizar caderno — propaga melhorias do caderneiro para um caderno existente
C) 🛠️ Modificar caderno — altera configurações de um caderno existente

Após a seleção, leia o arquivo correspondente em `instrucoes/` e execute a operação:
- A) `instrucoes/criar-caderno.md`
- B) `instrucoes/atualizar-caderno.md`
- C) `instrucoes/modificar-caderno.md`
